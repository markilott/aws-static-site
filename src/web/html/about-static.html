Static Page 2 - About

Front End Web Site. The site is hosted in S3 as a static site.

The web front end is built from vanilla HTML and JavaScript, styled using Bootstrap, bundled using Parcel, and uploaded to S3 using AWS CDK.

Back End Structure. Serverless on the AWS cloud.

The S3 hosted site is served via the CloudFront CDN providing SSL and the custom domain name. API Gateway and Lambda provide the register API with user data stored in DynamoDB.
